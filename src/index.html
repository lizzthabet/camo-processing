Speculative camouflage generator

A prototype for a wearable pattern generator that builds camouflage-inspired designs

Right now, this tool only generates a single design: a pixelated fractal noise pattern (generally called a multiscale pattern), using the colors from an uploaded image. In the future, it may expand to generate shape-disruptive patterns, too. The design for the noise pattern is loosely based on a scientific paper published in 2013. This tool's purpose is to generate large images of patterns that can be used for textiles, clothing, and other surfaces.

When a pattern is successfully generated, you can double-click on any canvas to save the outputted images.
